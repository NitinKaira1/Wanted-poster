# 🤠 Wanted Poster Cam

Detects your face via webcam and generates a hilarious criminal wanted poster.

## Setup

```bash
python -m venv venv
venv\Scripts\activate       # Windows
pip install -r requirements.txt
```

## Run (Flask GUI - recommended)

```bash
python app.py
```
Open browser → http://127.0.0.1:5000

## Run (Terminal only)

```bash
python main.py
```

## Groq AI Roasts (optional)

Get a free API key at https://console.groq.com  
Paste it into the browser UI — no code changes needed!
